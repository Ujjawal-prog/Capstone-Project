# Capstone-Project This is capstone project of Web developemnt.
Website Overview: 
The 92 Acres Platform is designed to help users search, view, and explore different property options such as apartments, villas, and flats in various cities.

Header Section:
At the top, the website displays the title “92 Acres Platform.”
Below the title, there are search and filter options:
Search property – to type and find specific properties.
Select location – to choose a city or area.
Property type – to filter by apartment, villa, flat, etc.
Dark mode button – to switch the website theme.

Footer Section:
The footer contains copyright information.
It mentions that the website is created for convenience for people interested in property finding.

Code and output:
The project was completely made by HTML and CSS.
To make body of the website we use HTML while CSS is used for colours and beautification.
